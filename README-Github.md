### Installation On Kali Linux


```bash
$ git clone https://github.com/Filza2/CVD23
$ cd CVD23
$ sudo bash Linux-installer.sh
$ python3 -m pip install -r requirments.txt
$ sudo python3 CVD-23.py
```
